Структура проекта

```
simple-os/
├── boot.asm          # Загрузчик
├── kernel.c          # Ядро на C
├── build.sh          # Скрипт сборки
├── grub.cfg          # Конфигурация GRUB
└── iso/              # Директория для ISO (создается автоматически) 
```

1. Загрузчик (boot.asm) - максимально простой

#### Инструкция по запуску

build.sh

```bash
#!/bin/bash
nasm -f bin boot.asm -o boot.bin
gcc -m32 -ffreestanding -c kernel.c -o kernel.o

ld -m elf_i386 -Ttext 0x100000 --oformat binary kernel.o -o kernel.bin
cat boot.bin kernel.bin > os-image.bin
dd if=/dev/zero of=os-image.bin bs=512 count=2880 seek=0 conv=notrunc
echo "Готово! Запустите: qemu-system-i386 -drive format=raw,file=os-image.bin"
```

2. Сделайте скрипт исполняемым и запустите:

```bash
chmod +x build.sh
./build.sh
```

3. Запустите в QEMU:

```bash
qemu-system-i386 -drive format=raw,file=os-image.bin
```


Объяснение максимально просто:

1. BIOS запускает наш код из boot.asm
2. boot.asm переключает процессор в 32-битный режим
3. kernel.c напрямую пишет в видеопамять по адресу 0xB8000
4. Видеокарта отображает содержимое видеопамяти на экране
Конфигурация GRUB (grub.cfg)

```cfg
set timeout=5
set default=0

menuentry "Simple OS" {
    multiboot /boot/kernel.bin
    boot
}
```

4. Скрипт сборки (build.sh)

```bash
#!/bin/bash

echo "=== Сборка Simple OS в ISO ==="

# Создаем директории
mkdir -p iso/boot/grub

# Компилируем загрузчик (теперь в ELF формате для GRUB)
echo "Компиляция boot.asm..."
nasm -f elf32 boot.asm -o boot.o

# Компилируем ядро
echo "Компиляция kernel.c..."
gcc -m32 -ffreestanding -nostdlib -c kernel.c -o kernel.o

# Линкуем ядро (ELF формат для GRUB)
echo "Линковка ядра..."
ld -m elf_i386 -Ttext 0x100000 -nostdlib boot.o kernel.o -o iso/boot/kernel.bin

# Проверяем Multiboot заголовок
echo "Проверка Multiboot..."
if grub-file --is-x86-multiboot iso/boot/kernel.bin; then
    echo "  Multiboot подтвержден"
else
    echo "  ОШИБКА: Multiboot не обнаружен!"
    exit 1
fi

# Копируем конфиг GRUB
cp grub.cfg iso/boot/grub/

# Создаем ISO образ
echo "Создание ISO..."
grub-mkrescue -o simple-os.iso iso

# Проверка размера
echo "Проверка ISO..."
if [ -f "simple-os.iso" ]; then
    echo "  ISO создан успешно: simple-os.iso"
    echo "  Размер: $(du -h simple-os.iso | cut -f1)"
else
    echo "  ОШИБКА: ISO не создан!"
    exit 1
fi

echo ""
echo "=== Готово! ==="
echo "Запуск: qemu-system-i386 -cdrom simple-os.iso"
echo "Запись на USB: dd if=simple-os.iso of=/dev/sdX bs=4M status=progress"
```

5. Альтернативный простой скрипт (build-simple.sh)

Если нужен еще более простой вариант без проверок:

```bash
#!/bin/bash

# Простая сборка ISO
mkdir -p iso/boot/grub

# Компиляция
nasm -f elf32 boot.asm -o boot.o
gcc -m32 -ffreestanding -nostdlib -c kernel.c -o kernel.o

# Линковка
ld -m elf_i386 -Ttext 0x100000 -nostdlib boot.o kernel.o -o iso/boot/kernel.bin

# Конфиг GRUB
cat > iso/boot/grub/grub.cfg << EOF
set timeout=5
menuentry "Simple OS" {
    multiboot /boot/kernel.bin
}
EOF

# Создание ISO
grub-mkrescue -o simple-os.iso iso

echo "ISO готов: simple-os.iso"
echo "Запуск: qemu-system-i386 -cdrom simple-os.iso"
```

#### Соберите ISO:

```bash
# Дайте права на выполнение
chmod +x build.sh

# Запустите сборку
./build.sh
```

3. Запустите в QEMU:

```bash
qemu-system-i386 -cdrom simple-os.iso
```

Или с отладкой:

```bash
qemu-system-i386 -cdrom simple-os.iso -debugcon stdio
```

4. Запись на USB (реальное железо):

```bash
# ВНИМАНИЕ: ЗАМЕНИТЕ /dev/sdX на ваш USB диск!
sudo dd if=simple-os.iso of=/dev/sdX bs=4M status=progress
sudo sync
```

Для UEFI загрузки (дополнительно):
В build.sh замените grub-mkrescue на:
```bash
grub-mkrescue --uefi -o simple-os.iso iso
```
